

Каталог овощной продукции на CMS 1С-Битрикс. 
Использованы компоненты: Каталог, Меню, Включаемая область, Цепочка навигации.
В компоненте Каталог применяется ЧПУ.
В шаблоне компонента Каталог использованы функции API 1С-Битрикс для получения 
изображений элементов раздела на странице списка разделов каталога.
